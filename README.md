## Escenario elegido
Análisis de Ventas

## Descripción
Proyecto realizado utilizando Git, GitHub, Jira y Google Colab.

El proyecto analiza datos simples de ventas y genera indicadores básicos junto con un gráfico.

## Estructura del proyecto

- /datos → datasets CSV
- /scripts → scripts Python
- /resultados → gráficos y resultados

## Ejecución

Ejecutar:

python scripts/analisis.py
